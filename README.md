## Perfect Pay

## **Tecnologias**

-   PHP
-   Laravel
-   Docker
-   PostgreSQL
-   Composer

## Como Rodar o Projeto

```sh
Copiar o .env-example para o .env
```

### **Execução**

```sh
# Para iniciar o servidor
$ docker-compose build
$ docker-compose up -d
$ composer install

# Para parar o servidor
$ docker-compose down
```
